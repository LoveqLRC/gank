# gank
布局代码参考自开源项目MeiZhi,仿干货集中营,该项目主要用于扎实自身技能、掌握Material Design控件和主流框架的使用。项目主要使用到的框架有SwipeBack,jiecaoVideoPlayer,retrofit,liteOrm等。
